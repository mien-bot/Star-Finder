# Phase 3: Polish

**Duration:** 2 weeks

## Goal
Production-ready features: multiple styles, export formats, editing tools

## Tasks

### 1. Rendering Styles
- [ ] Blueprint style (blue-white lines)
- [ ] Black & White style
- [ ] Architectural style (with materials)
- [ ] Minimal style (clean line art)
- [ ] Style selector in UI

### 2. Export Formats
- [ ] DWG export (via BlenderCAD addon or DXF conversion)
- [ ] DXF export (native Blender)
- [ ] PDF export (render to PDF)
- [ ] SVG export (from Blender or direct)
- [ ] PNG export (high-res renders)
- [ ] Blender file export (.blend)

### 3. Editing Tools
- [ ] Add/edit building walls
- [ ] Add doors and windows
- [ ] Dimension lines with auto-measure
- [ ] North arrow overlay
- [ ] Scale bar
- [ ] Labeling system

### 4. Auto-Scaling
- [ ] Read geotagged metadata from images
- [ ] Auto-calculate real-world dimensions
- [ ] Handle different coordinate systems

## Deliverables
- Multiple rendering styles
- Full export pipeline (DWG, DXF, PDF, PNG, blend)
- Basic editing capabilities
- Auto-scaling from geotagged images

## Success Criteria
- Can export to multiple formats including DWG/DXF
- Rendering style selection works
- Manual editing tools functional
